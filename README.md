『Seedeas』


概要
「Seedeas」は業種に関わらず、業務展開の機会を提供するサービス。簡単に言えば、異業種のマッチングサービス。自分のスキル（種）と誰かのアイデアが合わさった時に新たな可能性が生まれる、そんなサービスです。
